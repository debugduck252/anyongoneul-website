6. 음성 데이터 처리 정책

🎤 음성 원본은 서버에 영구 저장하지 않습니다. STT 변환 처리가 완료되면 즉시 삭제됩니다.

  • 음성 파일은 STT 처리를 위해 Firebase Functions를 통해 음성 인식 서비스에 일시 전송됩니다.
  • 전송된 음성 파일은 처리 완료 즉시 서버에서 삭제되며, 별도 저장하지 않습니다.
  • 음성 원본의 로컬 저장은 기본값이 OFF이며, 사용자가 명시적으로 허용한 경우에만 기기에 저장됩니다.
  • 로컬에 저장된 음성 파일은 설정 메뉴에서 언제든지 삭제할 수 있습니다.

7. 이용자의 권리 및 행사 방법

이용자는 안녕오늘에 대해 언제든지 다음 권리를 행사할 수 있습니다.

  • 열람 요청: 본인의 개인정보 처리 현황 확인
  • 정정 요청: 잘못된 개인정보의 수정
  • 삭제 요청: 개별 기록 삭제 또는 계정 탈퇴를 통한 전체 삭제
  • 처리 정지 요청: 동의 철회 또는 처리 중지

행사 방법

  • 앱 내 직접: 기록 상세 화면 → 삭제 / 설정 → 계정 탈퇴
  • 이메일 문의: support@anyongoneul.com

권리 행사 요청을 받은 날로부터 10일 이내에 처리 결과를 알려드립니다. 단, 법령에 따라 보존이 필요한 정보는 즉시 삭제되지 않을 수 있습니다.

8. 개인정보 파기 절차 및 방법

파기 절차

  • 개인정보는 보유 기간 만료 또는 처리 목적 달성 시 지체 없이 파기합니다.
  • 계정 탈퇴 후 7일 이내 철회가 없으면 30일 이내에 모든 데이터를 파기합니다.
  • 음성 원본은 STT 처리 완료 즉시 서버에서 파기합니다.

파기 방법

  • 전자적 파일: 복원이 불가능한 방법으로 영구 삭제
  • Firestore 데이터: 해당 문서 및 서브컬렉션 완전 삭제

9. 개인정보 보호 기술적·관리적 조치

  • 암호화 전송: 모든 데이터는 HTTPS(TLS 1.2 이상)로 전송됩니다.
  • 접근 제어: Firestore 보안 규칙을 통해 본인 데이터에만 접근 가능합니다. 구독, 사용량, 통계, 인사이트는 클라이언트가 직접 수정할 수 없습니다.
  • API 키 보호: 모든 외부 API 키는 서버(Firebase Functions)에서만 관리하며, 앱 코드에 포함하지 않습니다.
  • 결제 보안: 영수증 검증은 서버에서 Google Play Developer API를 통해 처리하며, 클라이언트 신뢰 없이 서버 검증만을 사용합니다.
  • 최소 권한 원칙: 서비스 제공에 필요한 최소한의 권한만 요청합니다.
